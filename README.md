# 한글 단어카드 - 우리 아이 첫 한글

2세 아동을 위한 한글 학습 단어카드 게임입니다.

## 특징

- **반응형 디자인**: iPhone 12 Pro(390×844px) 기준 한 화면에 4장의 카드 표시
- **플립 카드**: 이미지를 탭하면 한글 단어가 표시됨
- **3개의 단어 세트**:
  - 세트 1: 개, 고양이, 해, 나무
  - 세트 2: 물, 빵, 사과, 차
  - 세트 3: 달, 꽃, 별, 바다

## 실행 방법

`wordcard.html` 파일을 웹 브라우저에서 열면 됩니다. 별도의 서버 없이 로컬에서 동작합니다.

## 기술 스택

- HTML5, CSS3, Vanilla JavaScript
- 반응형 그리드 (2×2 레이아웃)
- SVG 인라인 일러스트
- 지브리 스타일의 부드러운 일러스트
